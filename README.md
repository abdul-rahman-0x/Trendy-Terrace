# 🛋️ Trendy Terrace: Premium Home Decor Showcase

Welcome to the **Trendy Terrace** project! This sleek and stylish website highlights the elegance and sophistication of our fictitious furniture and home decor brand. Below, you will find information on the project structure, technologies used, and how to set up and run the project.

## Project Overview


**Trendy Terrace** is a high-end, fully responsive e-commerce storefront designed for a boutique furniture brand. This project serves as a masterclass in Modern Web Layouts, focusing on high-precision typography, fluid responsiveness, and cinematic scroll interactions.
The website provides a seamless user journey from a captivating Hero section through curated collections and deep into product details, all while maintaining a consistent "Luxury Boutique" aesthetic.

## Technologies Used

- Structure: `HTML5` for semantic, SEO-friendly markup.
- Styling: `CSS3` featuring Custom Properties (Variables), Flexbox, and CSS Grid.
- Interactivity: `Vanilla JavaScript` for DOM manipulation and event handling.
- Animations: `ScrollReveal.js` for cinematic entrance effects.
- Iconography: `Font Awesome` for scalable, lightweight UI icons.

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/abdul-rahman-9/Trendy-Fully-Responsive-Website.git
    ```

2. Navigate to the project directory:

    ```sh
    cd Trendy-Fully-Responsive-Website
    ```

3. Open `index.html` in your web browser to view the website.
